== Description ==

A plugin that helps people share your content via powerful quotes. Provides an interface for sharing selected text directly to other websites along with a link to the original content. Also provides an interface on blockquote elements for sharing those quotes directly, and hooks to use the functionality throughout a custom theme.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Optional: Configure plugin options
4. Optional: Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= Can you add the option to share quotes on more social networks? =

I may add integration with certain websites or services as time goes on, but my main goal is to make sharing content simple and quick. Too many options could work counter to that goal.
